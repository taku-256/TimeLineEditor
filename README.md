# ロボコン タイムチャートエディタ (Robocon Timeline Editor)

競技ロボットの動作計画・タイムライン作成を行うための直感的なWebアプリケーションです。動画編集ソフトのような操作性と、ガントチャートのような工程管理を組み合わせたUIを提供します。

## 主な機能

* **高精度タイムライン**:
  - 0.1秒単位の時間軸スケール（デフォルト0〜120秒、変更可能）
  - マウスホイールによるズーム（カーソル中心）、Shift+ホイールでの横スクロール
  - スナップ設定（0.1s / 0.5s / 1s / OFF）の切り替え
* **レーン管理**:
  - 自由な追加・削除・名前変更
  - ドラッグ＆ドロップによる並び替え
* **ブロック編集**:
  - ダブルクリックによる直感的なブロック配置
  - ドラッグ移動（レーン間移動に対応）
  - 左右端ドラッグによるリサイズ（必須時間 `████████` とバッファ時間 `□□□□` を視覚的に区別表示）
  - プロパティパネルによる詳細な値編集（名前、色、開始時間、必須・バッファ時間、メモ）
* **イベント配置**:
  - ダイヤモンド型マーカーと縦線によるイベント定義（スタート、発射、得点など）
  - 得点（★）や発射（⚡）などの各種アイコンおよびカラー指定
* **便利ツール**:
  - Undo / Redo 機能 (Ctrl+Z / Ctrl+Y)
  - コピー＆ペースト (Ctrl+C / Ctrl+V)
  - 複数選択と一括削除 (Ctrl+クリック または 矩形ドラッグ選択)
  - インクリメンタルなブロック検索 (Ctrl+F)
  - プロジェクトのJSONファイル保存・読み込み
  - ダークモードとライトモードの切り替え

## 開発と実行手順

### 1. 依存関係のインストール
```bash
npm install
```

### 2. ローカル開発サーバーの起動
```bash
npm run dev
```
起動後、ブラウザで `http://localhost:5173/` にアクセスします。

### 3. 本番ビルド
```bash
npm run build
```
ビルド完了後、`dist/` ディレクトリに公開用ファイルが出力されます。

## GitHub Pages での公開手順

このプロジェクトは `vite.config.ts` でベースパスが `base: '/TimeLineEditor/'` に設定されており、GitHub Pages 上のサブディレクトリで正しく稼働するよう構成されています。

公開ページは以下のURLです：
[https://taku-256.github.io/TimeLineEditor/](https://taku-256.github.io/TimeLineEditor/)

### 公開の自動化 (GitHub Actions)
リポジトリの **Settings -> Pages -> Build and deployment -> Source** で **「GitHub Actions」** を選択した状態で `main` ブランチにプッシュすると、`.github/workflows/deploy.yml` により自動ビルドとデプロイが行われます。

### 方法1: ローカルから `gh-pages` コマンドで手動デプロイする場合
1. パッケージを追加：
   ```bash
   npm install -D gh-pages
   ```
2. `package.json` にスクリプトを追加：
   ```json
   "scripts": {
     "predeploy": "npm run build",
     "deploy": "gh-pages -d dist"
   }
   ```
3. 実行：
   ```bash
   npm run deploy
   ```

## 操作方法・ショートカットキー

| 操作 / ショートカット | 機能 |
|---|---|
| **ダブルクリック (空きレーン)** | 新規ブロックをスナップ位置に作成 |
| **ドラッグ (ブロック中央)** | ブロックの時間・レーン移動 |
| **ドラッグ (ブロック左右端)** | 必須時間（左側）およびバッファ時間（右側）の長さ調整 |
| **Ctrl + Z / Y** | 元に戻す (Undo) / やり直す (Redo) |
| **Ctrl + C / V** | 選択ブロック・イベントのコピー / 貼り付け |
| **Ctrl + D** | 選択ブロック・イベントの複製 |
| **Delete / Backspace** | 選択アイテムの削除 |
| **Ctrl + A** | すべてのアイテムを選択 |
| **Ctrl + F** | ブロック検索パネルを開く |
| **ホイールスクロール** | タイムラインの拡大 / 縮小 |
| **Shift + ホイールスクロール** | タイムラインの左右スクロール |
| **Ctrl + ホイールスクロール** | タイムラインの上下スクロール |
| **Esc** | 選択解除、検索やメニューを閉じる |
| **右クリック** | コンテキストメニューの表示 |
